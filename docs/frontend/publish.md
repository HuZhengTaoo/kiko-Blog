## 小程序管理后台
:::danger
   - 第一次发布并未出现要填写提供测试账号的选项,一般都会被打回。
   - 第二次的时候就会出现提供测试账号的入口，到时候填写可用的账号就行
   - 如果是发布正式版本的小程序(www域名下),需要提前准备相应的可测试账号以及相关课程条目等信息
:::
